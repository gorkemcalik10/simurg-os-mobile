SIMURG OS PREMIUM SHELL V1

Base:
- Built on SIMURG_OS_MOBILE_STABLE_BASELINE_V1.
- Mobile-first premium shell update.
- Desktop layout is intentionally not redesigned.

Main changes:
1. Ana / Home
- Reworked as a premium daily decision center.
- Includes Good Morning header, recovery hero card, Sleep / HRV / RHR mini cards, today's training card, Coach note and Start Workout button.
- Added Simurg Alert button on the right side of the header.

2. Simurg Alert
- Opens a compact daily alert panel.
- Shows today's decision, recovery signal and sleep note.
- Designed to later include deload, risk, missing Polar data and coach warnings.

3. Bottom Navigation
- Changed mobile bottom navigation to 6 tabs:
  Ana, Recovery, Gym, Logger, Polar, Menü.

4. Polar Tab
- Added a new Polar tab as a clean placeholder / summary screen.
- Built for future Polar Loop Gen 2 / Polar Flow data review.
- Current screen summarizes intended data areas: Nightly Recharge, HRV, Sleep, Training Load.

5. Menu
- Menu sheet now contains only:
  Coaching, Program, Weekly, Monthly, Data Center.

Notes:
- Recovery, Logger and Gym data engines were not rewritten in this version.
- Gym remains simple and manual-entry focused.
- Logger remains the session summary / correction area.
- Polar integration will be shaped after real Polar Flow export data is checked.
