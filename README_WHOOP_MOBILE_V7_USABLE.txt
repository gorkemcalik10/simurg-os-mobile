SIMURG OS — WHOOP Mobile V7 Usable System

Bu sürüm V5 full system üzerine hazırlanmıştır.

Eklenenler:
- Mobil alt menü sadeleştirildi: Ana / Recovery / Logger / Gym / Menü.
- Menü butonu tüm Simurg modüllerini açan bottom-sheet paneli gösterir.
- Gym Mode mobilde tek kolon, büyük inputlar, sticky save aksiyonları ve daha kullanılabilir kart yapısına getirildi.
- Workout Logger mobilde Seans Kontekst kartını üstte gösterir; Apple Watch verisi korunur.
- Ana ekran WHOOP mantığına yakın: Bugünün kararı, readiness, recovery, sleep, HRV, fiziksel yük.
- Polar recoveryEntries ve Universal Import polar_recovery desteği korunur.
- Desktop sol menü korunur; mobilde gizlenir.

Test önerisi:
1. Mobilde Ana, Recovery, Logger, Gym sekmelerini kontrol et.
2. Menü > Daily / Weekly / Analytics / Data modüllerini aç.
3. Gym Mode'da kg/rep gir, Save yap, Logger ve Daily Summary'de kayıtların geldiğini kontrol et.
4. Data Center > Örnek Polar JSON ile polar_recovery import test et.
